Summary Latitude Weather

The initial part of the project created a Python script to analyze and visualize the weather of 500+ random cities across the world of varying distance from the equator for the report date. The project used utilizing a Python library Citipy and the OpenWeatherMap API to create a representative model of weather across world cities.

After establishing the dataset, Matplotlib was utilized to perform 4 different scatter plots. Specifically looking at max temperature, cloudiness, humidity, and wind speed for each city latitude.
